# Mini-Project_C
A neat little C program created for practice and learning.
A simple terminal-based Tic Tac Toe game written in C, played between two players. Players take turns to enter the position on the grid to place their X or O. The game continues until one player wins or the grid is full (a draw).


---

FEATURES:

2-player game (no AI, just two people taking turns)

Input the position number to place X or O

Displays the game board after each move

Detects win or draw automatically



---

HOW TO PLAY:

1. Compile the code using:

gcc tic_tac_toe.c -o tic_tac_toe


2. Run the game:

./tic_tac_toe


3. Players take turns entering a number (1-9) corresponding to the grid position:



1 | 2 | 3
---------
4 | 5 | 6
---------
7 | 8 | 9

4. The first player uses X and the second player uses O.


5. The game announces the winner or a draw at the end.
